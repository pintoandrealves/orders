# Projeto Java 17 com spring boot 2.7

Consulta de clientes e consulta de dados de Compras

## 🚀 Início.

O principal objetivo do projeto é integrar várias tecnologias em uma aplicação de microserviço Spring boot.

## 🛠 Tecnologias.

<ol>
  <li>Java</li>
  <li>Spring boot</li>
  <li>Mockito</li>
  <li>Junit</li>
  <li>log4j2</li>
  <li>Swagger</li>
  <li>Insomnia</li>
</ol>
